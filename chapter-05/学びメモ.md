## React フラグメント

↓ みたいに、2 階層になっているとエラーになるので親要素で囲んであげる必要がある。

```react
function Cat({ name }) {
        return (
            <h1>Hello, cat's name is {name}</h1>
            <p>He's good</p>
        );
      }
```

正解

```react
function Cat({ name }) {
        return (
            <>
                <h1>Hello, cat's name is {name}</h1>
                <p>He's good</p>
            </>
        );
      }
```
